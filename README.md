## Sales Performance Report

### Project Overview

This project presents an interactive Sales Performance Dashboard developed in Power BI to analyze business performance across multiple countries, product categories, and time periods. The dashboard provides insights into revenue, profit, customer behavior, market share, return rates, and product performance to support data-driven business decisions.

### Key Features:

Interactive KPI tracking dashboard
Revenue and profit analysis
Product category performance analysis
Country-wise market share visualization
Monthly sales trend analysis
Top-performing products identification
Customer and order analysis
Return rate and profitability monitoring
Dynamic filtering by:
- Date Range
- Country
- Year

### Report Insights:

- Total Revenue: 24.91M
- Total Profit: 10.46M
- Total Orders: 84K
- Profit Margin: 41.97%
- Total Customers: 17K

### Major Findings:

- Bikes category contributes the highest revenue.
- United States and Australia are the leading markets.
- Mountain-200 product series are top-selling products.
- Revenue peaks during the first half and year-end periods.
- Return rates remain low, indicating strong customer satisfaction.

### Tools & Technologies Used:

- Power BI Desktop
- DAX (Data Analysis Expressions)
- Power Query
- Data Modeling
- Interactive Visualizations
